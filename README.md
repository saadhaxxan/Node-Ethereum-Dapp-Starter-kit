##  INSTALLATION

```
git clone https://github.com/saadhaxxan/Node-Ethereum-Dapp-Starter-kit.git
cd Node-Ethereum-Dapp-Starter-kit
npm install
npm install -g truffle
npm install -g web3
truffle compile
truffle migrate
```
